## Contributing

Thank you for considering contributing to our project! Contributions are welcome from everyone.

### Reporting Issues
If you encounter any issues with the project, please report them on the [GitHub Issues](https://github.com/H3nryK/Portfolio/issues) page. When reporting issues, please provide as much detail as possible, including steps to reproduce the issue, expected behavior, and screenshots if applicable.

### Submitting Pull Requests
We welcome contributions in the form of pull requests to help improve the project. Before submitting a pull request, please ensure the following:

1. **Fork the Repository:** Fork the repository to your GitHub account and create a new branch for your changes.

2. **Make Changes:** Make your desired changes in the new branch. Ensure that your changes adhere to the project's coding conventions and standards.

3. **Test Changes:** Test your changes locally to ensure they work as expected and do not introduce any new issues.

4. **Commit Changes:** Commit your changes with descriptive commit messages that explain the purpose of each change.

5. **Push Changes:** Push your changes to your forked repository.

6. **Create Pull Request:** Once you've pushed your changes, go to the GitHub repository and create a pull request from your branch to the main branch of the original repository.

7. **Describe Pull Request:** Provide a detailed description of your changes in the pull request, including the problem solved and any relevant context.

8. **Review Process:** Your pull request will be reviewed by project maintainers, and feedback may be provided. Please be responsive to any comments or questions during the review process.

9. **Merge Pull Request:** Once your pull request has been approved, it will be merged into the main branch of the project.

Thank you for your contributions!
